rooftop-admin-theme is the plugin used to make WordPress look like Rooftop CMS

== Description ==

This plugin switches off some sidebar options, removes WP functionality related to theme-based websites, and tidies things up for RoofTop.

Track progress, raise issues and contribute at http://github.com/rooftopcms/rooftop-admin-theme

== Installation ==

rooftop-admin-theme is a Composer plugin, so you can include it in your Composer.json.

Otherwise you can install manually:

1. Upload the `rooftop-admin-theme` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. There is no step 3 :-)

== Frequently Asked Questions ==

= Can this be used without Rooftop CMS? =

Yes, it's a Wordpress plugin you're welcome to use outside the context of Rooftop CMS. We haven't tested it, though.
